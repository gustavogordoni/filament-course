> Projeto de estudos desenvolvido com base na playlist do YouTube do canal **Alexandre Cardoso**.  
> Este repositório acompanha o aprendizado prático do **FilamentPHP**, um painel administrativo para aplicações Laravel.

---

## Sobre o Projeto

Este repositório é **exclusivamente para fins de estudo**. O conteúdo foi desenvolvido seguindo as aulas da playlist disponível no YouTube, reproduzindo e adaptando os exemplos ensinados ao longo do curso.

Trata-se de um ambiente de aprendizado e prática com o ecossistema **Laravel 12 + FilamentPHP 5**.
